User Acquisition Funnel & Conversion Analytics Dashboard
Tools: Tableau, Data Visualization, Funnel Analysis

- Designed a multi-view Tableau dashboard tracking the full user journey from registration to first payment across 8,460 users, surfacing step-level drop-off rates (Registration → Trial: 23%; Trial → Payment: 6.8%) to identify highest-impact conversion bottlenecks.
- Built an inverted funnel visualization displaying both absolute user counts and cumulative conversion rates at each onboarding stage, enabling quick identification of where users disengage in the activation flow.
- Developed a dual-axis combo chart overlaying monthly registration volume with registration-to-trial conversion rate trends, revealing seasonal patterns and cohort-level performance shifts across a 6-month period.
- Implemented cross-chart interactivity using Tableau actions, enabling the monthly registrations chart to act as a dynamic filter - selecting a specific month or data point instantly updates all KPI blocks and the funnel view to reflect period-specific metrics, supporting deeper ad hoc exploration without additional filtering steps.
